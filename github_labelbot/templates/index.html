GitHub Labelbot sample webhook

This is a GitHub webhook page serving as a sample for GitHub LabelBot project. It allows GitHub to notify LabelBot about new issues or issue comments in real time. The Bot then scans the issues and assigns new labels if necessary according to configured rules.

Configuration

To allow LabelBot to label issues in your repository, additional steps are required:

  • Add your GitHub bot user as an collaborator to the project you want to label.
  • Create a new webhook in your repo with e.g. {{ url_for('hook', _external=True, _scheme='https') }} Payload URL and issues and issue comment events allowed.
  • Set WEBHOOK_TOKEN environment variable corresponding to the secret token you used during token creation.
